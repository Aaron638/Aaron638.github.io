Projects:

  • Noted Webapp

    Spring 2020 - SE 319 Construction of User Interfaces

    A web application that acted as our own take on Google Keep. Users can enter text notes, which are then put into cards. These notes are saved locally on their machine. A set of javascript scripts run in the page to categorize notes based off of keywords. For example, when a user enters "Buy eggs, milk, and ham tomorrow.", a list is shown of what the user needs to buy. Another example is the map card. When a user checks a box, the user text is sent to a reverse geocoding API which then displays a map on the card.

  • Hide and Seek App

    Fall 2019 - COM S 309 Software Development Practices

    An Android game application which allows multiple players to play a game of hide and seek in a larger area using GPS. Using the Android Google Maps API, the hiders can see players on the same team, and react accordingly. A code is inputted by the seekers when a hider has been found. This code is cross checked with the position of both players by the server to prevent cheating. This was a learning project where we used Git with CI/CD, Android, and Spring Boot. Android Volley and websockets were used to connect to the backend.
